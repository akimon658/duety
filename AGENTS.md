# Duety

This service synchronizes assignment information from a university LMS calendar
URL to Google Tasks.

## Tech Stack

- Built with **Deno**, **Fresh v2**, **Vite**, and **daisyUI**.

## Authentication

- Authentication is handled via a proxy server.
- Local development uses Caddy to replicate the authentication flow.
